## 第 0 章 阅读导引

### 0.1 AGMIMM 是什么？

**自适应高斯混合交互多模型（Adaptive Gaussian Mixture Interacting Multiple Model, AGMIMM）** 是一种面向 **非线性跳跃马尔可夫系统（JMS）** 的贝叶斯滤波器，专门用于 **地月空间非合作机动 CSO** 的 **仅角度（angles-only）多传感器跟踪**。

核心思想可以拆成三层：

1. **IMM 框架**：目标模式（弹道 / 机动）未知，用马尔可夫链描述随机切换；各模式维护条件 pdf。
2. **GMM 表示**：每个模式下的 pdf 用高斯混合（GM）近似，以刻画混沌 CR3BP 动力学产生的 **强非高斯** 不确定性。
3. **自适应 AGM**：在动力学非线性、测量非线性、FoV 边界、负信息（未检测）和 Jacobi 约束等区域 **动态分裂** 混合分量；在模式交互混合步用 **GMRC 聚类合并** 控制计算量。

与现有方法相比，AGMIMM 的两项关键创新是：

- 相对一般 AGM 滤波器：**额外适应随机模式切换**（JMS/IMM 结构）。
- 相对 IMM/GMIMM：**多准则分裂 + 基于聚类的混合后验合并**，而非单高斯或按权重裁剪合并。

**【文字说明】** 可以把 AGMIMM 理解成在 **三条轴** 上同时做自适应：(1) **模式轴**——IMM 交互与模式概率更新；(2) **表示轴**——GMM 分量数通过分裂增、通过 GMRC 减；(3) **信息轴**——正检测、负检测、Jacobi/碰撞约束分别对应不同分裂或剪枝准则。一般 AGM 文献多关注 (2)(3) 中的动力学与测量非线性；本文 **首次** 在 JMS/IMM 框架下系统整合 GMRC 与多准则 AGM，并应用于地月仅角度多传感器跟踪。

### 0.2 一句话对比

| 方法 | 状态表示 | 模式切换 | 负信息 / FoV | 非高斯适应 |
|------|----------|----------|--------------|------------|
| EKF / SRUKF | 单高斯 | 通常单模型 | 难系统处理 | 无 |
| IMM | 各模式单高斯 | 马尔可夫 IMM | 未专门处理 | 弱 |
| GMIMM | 各模式 GMM | IMM + GMM | 未专门处理 | 部分（固定分量） |
| **AGMIMM** | 各模式 GMM | IMM + **GMRC 合并** | **RFS + 递归分裂** | **多准则 AGM 分裂** |

## 第 1 章 问题 formulation 与动机

### 1.1 为什么地月 CSO 跟踪特别难？

**直觉链条**：

1. **动力学**：CR3BP 是强非线性、混沌系统 → 即使目标弹道飞行，长时间传播后 pdf 也 **高度非高斯**。
2. **观测**：地月空间监视受月球遮挡、光照、月球排除角限制 → **长观测间隙** 常见；间隙内不确定性快速增长。
3. **机动**：非合作目标点火时刻、持续时间、幅值未知 → 需 **JMS** 同时估计状态与模式。
4. **传感器**：本文考虑 **仅角度** 的空间基观测网 → 测量函数强非线性；未检测时含 **负信息**。

因此，单高斯 IMM/EKF 或固定分量的 GMIMM 在 **观测间隙 + 近月机动** 场景下容易丢失 track custody。

**【文字说明】** 地月空间域感知（SDA）的核心任务之一是 **维持 track custody**——即在长时间内持续掌握目标状态及其不确定性。与合作目标不同，非合作 CSO 不会广播辅助信息；监视又常依赖 **被动角度测量**，可观测性弱、非线性强。论文指出：即便目标按弹道飞行，由于 CR3BP 的混沌性，状态 pdf 也会迅速偏离高斯；若再叠加未知机动，传统以单高斯 IMM 为核心的滤波器往往在 **第一次长观测间隙** 后就无法恢复。AGMIMM 的设计目标，正是在这种“动力学混沌 + 观测稀疏 + 模式未知”的三重困难下仍能给出 **可信的非高斯 pdf**。

### 1.2 状态、动力学与 Jacobi 常数

**式 (1)** — 状态定义（会合系 / synodic frame，相对地月质心）：

$$
x_k = [r_k^{\top}, v_k^{\top}]^{\top} = [x_k, y_k, z_k, \dot{x}_k, \dot{y}_k, \dot{z}_k]^{\top}
$$

**式 (2)–(3)** — 归一化单位：1 [LU] = 379632.95 km，1 [TU] = 368232.68 s。

**式 (4)–(7)** — CR3BP 连续动力学（模式相关加速度 $w(t,\tau)$）：

$$
\ddot{x} - 2\dot{y} = w_x(t,\tau) + \frac{\partial U}{\partial x}\Big|_{r}, \quad \ddot{y} + 2\dot{x} = w_y(t,\tau) + \frac{\partial U}{\partial y}\Big|_{r}, \quad \ddot{z} = w_z(t,\tau) + \frac{\partial U}{\partial z}\Big|_{r}
$$

伪势 $U(r)$ 见式 (7)。当 $w=0$ 时为经典 CR3BP。

**式 (8)** — Jacobi 常数（弹道积分量）：

$$
C(r_k, v_k) = 2U(r_k) - \|v_k\|^2
$$

**物理含义**：同一弹道轨迹上 $C$ 恒定；混合分量内 $C$ 的方差大 → 分量内包含不同轨道族 → 线性化传播误差大（见第 5.2 节）。

**【文字说明】** Jacobi 常数 $C$ 是 CR3BP 弹道运动的积分量，可理解为“等效能量”指标：同一轨道上 $C$ 不变，不同轨道则 $C$ 不同。对 GMM 滤波而言，若某个高斯分量内部 $C$ 的散布很大，说明该分量 **同时覆盖多条不同轨道**；用线性化（或统计线性化）把这样一个“胖分量”一次性传播，必然产生较大误差。因此 $C$ 的方差可作为 **是否需要分裂** 的廉价判据——这是 AGMIMM 相对一般 AGM 在地月问题上的专门设计。

### 1.3 模式转移（JMS）

**式 (9)** — 齐次马尔可夫链：

$$
\pi_{ij} = \Pr(\tau_k = j \mid \tau_{k-1} = i)
$$

假设：转移概率已知、时不变、与连续状态无关（算例中可故意违背以测鲁棒性）。

模式 **左连续**： $\tau_k$ 在 $t \in (t_{k-1}^+, t_k^-]$ 内有效。

### 1.4 多传感器仅角度测量与 RFS

每个观测星 $o$ 有 body frame $B^{(o)}$，相机视轴沿 $\hat{b}_2^{(o)}$。

**式 (11)–(13)** — 检测概率与角度测量：

$$
p_D^{(o)}(x; S^{(o)}) = I_{S^{(o)}}(r) \times \psi(x, \xi^{(o)})
$$

$$
h^{(o)}(x) = [\arctan(\hat{u}^{(o)\top}\hat{b}_1^{(o)}/\hat{u}^{(o)\top}\hat{b}_2^{(o)}), \arcsin(\hat{u}^{(o)\top}\hat{b}_3^{(o)})]^{\top}
$$

其中 $\hat{u}^{(o)} = (r - r_{\text{obs}}^{(o)}) / \lVert r - r_{\text{obs}}^{(o)} \rVert$。

**式 (15)** — 多传感器测量为元组 $Z_k = (Z_k^{(1)}, \ldots, Z_k^{(O)})$；每个 $Z_k^{(o)}$ 为 **Bernoulli RFS**：要么空集 $\emptyset$（未检测），要么含单个测量向量。

**动机**：RFS 似然把 **检测概率 + 测量噪声 + 未检测** 统一进 Bayes 更新，使负信息可系统融合（第 4 章详述）。

**【文字说明】** 地月监视中，“没有量测”并不等于“没有信息”。若某观测星的 FoV 覆盖某区域而 **未检测到** 目标，则目标位于该区域内的概率应下降——这称为 **负信息（negative information）**。Bernoulli RFS 把“检测到一点”与“检测为空集”统一写成随机有限集似然；检测概率 $p_D(x)$ 随目标位置、光照、遮挡、月球排除角变化，因而是 **状态相关** 的。AGMIMM 后续对 $p_D$ 的 Taylor 近似与 FoV 边界分裂，都是为了在 GMM 框架下 **正确吸收** 这类信息。

### 1.5 离散 JMS 模型与 Bayes 链

**式 (16)–(17)**：

$$
x_k = f(x_{k-1}) + w_{k-1}(\tau_k), \quad z_k^{(o)} = h^{(o)}(x_k) + \nu_k^{(o)}(\tau_k)
$$

**式 (18)** — 递推链：

$$
\cdots \to p_{k-1}(\tilde{x}_{k-1} \mid Z_{0:k-1}) \to p_{k \mid k-1}(\tilde{x}_k \mid Z_{0:k-1}) \to p_k(\tilde{x}_k \mid Z_{0:k}) \to \cdots
$$

## 第 2 章 非线性 JMS Bayes 滤波（Sect. 3.1）

### 2.0 动机、背景与符号（docx §3.1）

在跳跃马尔可夫系统（JMS）中，目标运动模式（如“弹道飞行”或“机动推进”）会随时间随机切换。我们不仅要估计连续状态（位置、速度），还要推断当前离散模式 τ。两者耦合：不同模式对应不同动力学、过程噪声乃至量测模型。

**非线性 JMS 贝叶斯滤波目标**：递归计算联合状态的后验 pdf $p_{k \mid k}(\tilde{x}_k \mid Z_{0:k})$，其中 $Z_{0:k}$ 为直到 $t_k$ 的全部多传感器量测。

**三方面挑战**：

1. **非线性**：动力学 $f$ 与量测 $h$ 均非线性（本文 CR3BP）。
2. **模式切换**：τ 服从马尔可夫链，联合后验混合成分数 **指数增长**。
3. **多传感器与随机检测**：各传感器可能漏检，需用 RFS 统一处理“未检测”带来的 **负信息**。

第 3.1 节给出 **精确** 贝叶斯递推，为 IMM / GMIMM / AGMIMM 等子最优近似提供理论基准。

**核心符号与假设**（与第 0.3 节符号表一致）：

- 连续状态 $x_k \in \text{R}^6$（位置+速度）。
- 离散模式 $\tau_k \in \{1,\ldots,M\}$，转移概率 $\pi_{ij}$ 已知且时不变。
- 多传感器量测 $Z_k = (Z_k^{(1)}, \ldots, Z_k^{(O)})$；每个 $Z_k^{(o)}$ 为 Bernoulli RFS（空集或未检测到 / 单点量测）。
- **联合量测似然**（条件独立）： $g_k(Z_k \mid \tilde{x}) = \prod_o g_k^{(o)}(Z_k^{(o)} \mid \tilde{x})$。

### 2.1 联合后验与多传感器似然分解

**式 (19)** — Bayes 更新：

$$
p_{k \mid k}(\tilde{x} \mid Z_{0:k}) = \frac{g_k(Z_k \mid \tilde{x})\, p_{k \mid k-1}(\tilde{x})}{\int \sum_{\ell=1}^{M} g_k(Z_k \mid \tilde{x}^{(\ell)})\, p(\tilde{x}^{(\ell)})\, d\tilde{x}}
$$

**式 (20)** — 多传感器似然可分解为乘积（条件独立）：

$$
g_k(Z_k \mid \tilde{x}) = \prod_{o=1}^{O} g_k^{(o)}(Z_k^{(o)} \mid \tilde{x})
$$

**式 (21)–(22)** — 更新可写为 **单传感器算子** 的复合：

$$
p_{k \mid k}^{(\tilde{x}^{(j)})}(\tilde{x}_k \mid Z_{0:k}) = \Psi_k^{(O)} \circ \cdots \circ \Psi_k^{(1)}\, p_{k \mid k-1}^{(\tilde{x}^{(j)})}(\tilde{x}_k \mid Z_{0:k-1})
$$

$$
[\Psi_k^{(o)} p](\tilde{x}) = \frac{g_k^{(o)}(Z_k^{(o)} \mid \tilde{x})\, p(\tilde{x})}{\int \sum_{\ell} g_k^{(o)}(Z_k^{(o)} \mid \tilde{x}^{(\ell)})\, p(\tilde{x}^{(\ell)})\, d\tilde{x}}
$$

**推导要点**：先对联合似然因式分解，再逐传感器做 Bayes 更新；每步归一化分母含对所有模式的积分，保证概率守恒。

**【docx 文字说明 — 式 (19)】** 标准 Bayes 公式的联合版本：先验 $p_{k \mid k-1}(\tilde{x})$ 与当前量测似然 $g_k(Z_k \mid \tilde{x})$ 相乘后归一化。分母对 **所有连续状态** 与 **所有模式** 积分/求和。为显式区分模式，记 $\tilde{x}^{(j)} = [x_k^{\top}, \tau_k=j]^{\top}$。

**【docx 文字说明 — 式 (20)】** 给定目标状态与模式，各传感器观测 **条件独立**（仅通过 x_k 耦合）。乘积结构使多传感器更新可 **顺序执行** 单传感器更新，极大简化计算。

**【docx 文字说明 — 式 (21)–(22)】** 量测条件独立时，可按任意顺序依次用第 $o$ 个传感器的似然更新当前密度： $\Psi_k^{(o)}$ 将已融合前 $o-1$ 个传感器信息的密度，与第 $o$ 个传感器似然相乘并归一化。最终等价于同时使用全部传感器。

### 2.2 预测步

**式 (23)** — Chapman–Kolmogorov + 全概率：

$$
p_{k \mid k-1}(\tilde{x}_k^{(j)} \mid Z_{0:k-1}) = \int \sum_{i=1}^{M} p_{k \mid k-1}(\tilde{x}_k^{(j)} \mid \tilde{x}_{k-1}^{(i)})\, p_{k-1}(\tilde{x}_{k-1}^{(i)} \mid Z_{0:k-1})\, d\tilde{x}_{k-1}
$$

**复杂度障碍**：精确解需 $O(M^k)$ 项（模式历史指数增长）+ 非线性动力学/测量 → 必须近似（IMM 族）。

**【docx 文字说明 — 式 (23)】** 预测步结合两个随机过程：

1. **状态转移密度** $p_{k \mid k-1}(\tilde{x}_k^{(j)} \mid \tilde{x}_{k-1}^{(i)})$：由模式转移 $\pi_{ij}$、动力学 $f$、过程噪声（高斯）共同决定。
2. **上时刻后验** $p_{k-1}(\tilde{x}_{k-1}^{(i)} \mid Z_{0:k-1})$。

对上一时刻 **所有可能** 的连续状态与 **所有模式** 积分/求和，得当前先验。模式 τ 不可直接观测且与连续状态耦合，预测必须考虑全部模式历史——这正是 JMS 复杂度指数增长的根本原因。

### 2.3 模式分解与 mode-conditioned 方程

定义 mode-conditioned pdf：

**式 (24)–(25)**：

$$
p_{k \mid k-1}^{(j)}(x_k) = p_{k \mid k-1}(x_k \mid \tau_k=j, Z_{0:k-1}), \quad p_k^{(j)}(x_k) = p_k(x_k \mid \tau_k=j, Z_{0:k})
$$

**式 (26)–(27)** — 模式概率：

$$
\mu_{k-1}^{(i)} = \Pr(\tau_{k-1}=i \mid Z_{0:k-1}), \quad \bar{\mu}_{k \mid k-1}^{(j)} = \sum_{i=1}^{M} \pi_{ij}\, \mu_{k-1}^{(i)}
$$

**式 (28)–(29)** — 单传感器对 mode-conditioned 密度与模式概率的更新算子 $\Psi_k^{(o,j)}$。

**【docx 文字说明 — 式 (28)–(32)】** 对固定模式 $j$， $\Psi_k^{(o,j)}$ 同时作用在模式条件密度 $p^{(j)}(x)$ 与模式先验概率 $\bar{\mu}^{(j)}$ 上：

- $g_k^{(o,j)}(Z_k^{(o)} \mid x)$：模式 $j$ 下的量测似然。
- $p_k^{(o,j)}(Z_k^{(o)} \mid Z_{0:k-1})$：模式 $j$ 的 **预测似然（边际）**。

模式条件密度更新的分母需对 **所有模式** 求和——量测可能来自任一模式，模式竞争通过分母中的混合体现。模式概率更新则是 Bayes 公式直接用于离散随机变量。

**式 (32)–(35)** — 核心 IMM 递推：

$$
p_{k \mid k-1}^{(j)}(x_k) = \int p_{k \mid k-1}(x_k \mid x_{k-1}, \tau_k=j)\, p_{k-1}(x_{k-1} \mid \tau_k=j, Z_{0:k-1})\, dx_{k-1}
$$

$$
p_k^{(j)}(x_k) = \Psi_k^{(O,j)} \circ \cdots \circ \Psi_k^{(1,j)}\, p_{k \mid k-1}^{(j)}(x_k \mid Z_{0:k-1})
$$

$$
\mu_k^{(j)} = \Psi_k^{(O,j)} \circ \cdots \circ \Psi_k^{(1,j)}\, \bar{\mu}_{k \mid k-1}^{(j)}
$$

**式 (34)** — **混合 mode-conditioned 后验**（IMM 交互步的关键）：

$$
p_{k-1}(x_{k-1} \mid \tau_k=j, Z_{0:k-1}) = \sum_{i=1}^{M} \frac{\pi_{ij}\, \mu_{k-1}^{(i)}}{\bar{\mu}_{k \mid k-1}^{(j)}}\, p_{k-1}^{(i)}(x_{k-1})
$$

**直觉**：预测到 $t_k$ 时模式为 $j$，但 $t_{k-1}$ 的真实模式可能是 $i$；上式按转移概率 $\pi_{ij}$ 与后验模式权重 $\mu_{k-1}^{(i)}$ 做 **交互混合**。

**【docx 文字说明 — 式 (33)–(34)】** 将预测步 (23) 写成模式条件形式：式 (32) 中 $p_{k-1}(x_{k-1} \mid \tau_k=j, Z_{0:k-1})$ 即 **混合后验**。要预测当前模式 $j$ 下的状态，并不知道 $t_{k-1}$ 目标处于哪个模式，因此把上时刻各模式后验 $p_{k-1}^{(i)}(x_{k-1})$ 按 $\pi_{ij}$ 与 $\mu_{k-1}^{(i)}$ 加权混合并归一化——这就是当前模式 $j$ 下动力学传播的 **初始分布**。

权重 $\pi_{ij} \mu_{k-1}^{(i)} / \bar{\mu}_{k \mid k-1}^{(j)}$ 是一种 **贝叶斯回溯**：给定当前模式 $j$，上一时刻模式为 $i$ 的后验概率。这正是 IMM **混合（mixing）** 步骤的精确表达。

### 2.4 式 (34) 的推导要点

**目标**：在 $t_k$ 时刻 **假设** 当前模式为 $j$，构造用于预测的模式条件先验 $p(x_{k-1} \mid \tau_k=j, Z_{0:k-1})$。

**Step 1 — 全概率（对 $t_{k-1}$ 的模式求和）**：

$$
p(x_{k-1} \mid \tau_k=j, Z_{0:k-1}) = \sum_{i=1}^{M} p(x_{k-1} \mid \tau_{k-1}=i, \tau_k=j, Z_{0:k-1})\, \Pr(\tau_{k-1}=i \mid \tau_k=j, Z_{0:k-1})
$$

**Step 2 — 标准 IMM 近似**：给定 $\tau_k=j$， $\tau_{k-1}$ 与 $\tau_k$ 在给定 $Z_{0:k-1}$ 下 **近似独立**，故

$$
\Pr(\tau_{k-1}=i \mid \tau_k=j, Z_{0:k-1}) \approx \Pr(\tau_{k-1}=i \mid Z_{0:k-1}) = \mu_{k-1}^{(i)}
$$

**Step 3 — 马尔可夫转移**： $p(x_{k-1} \mid \tau_{k-1}=i, \tau_k=j, \cdot) \approx p_{k-1}^{(i)}(x_{k-1})$，权重由 $\pi_{ij}$ 给出。

**Step 4 — 归一化**：分母 $\bar{\mu}_{k \mid k-1}^{(j)} = \sum_i \pi_{ij} \mu_{k-1}^{(i)}$ 保证混合 pdf 积分为 1。

> **物理直觉**：IMM 交互步 = “若下一步是机动模式，上一步状态更可能来自各历史模式的加权混合”，而非硬切换。

**【docx 文字说明 — 模式条件分解】** 联合分布可分解为 $M$ 个并行的模式条件滤波器：每个负责一种模式下的连续状态密度 $p_k^{(j)}(x_k)$，模式概率 $\mu_k^{(j)}$ 则根据各模式与量测的匹配程度更新。这是 IMM 类算法的结构基础。

### 2.5 精确 JMS 递推结构小结

```text
输入：p_{k-1}^{(j)}(x), mu_{k-1}^{(j)}
  ↓ 混合：对每个 j，计算混合后验 (34)
  ↓ 预测：对每个 j，传播得 p_{k \mid k-1}^{(j)} (32)
  ↓ 更新：依次用 Psi_k^{(o,j)} 更新密度与模式概率 (28-29, 35)
输出：p_k^{(j)}(x), mu_k^{(j)}
```

**为何需要近似？** 式 (34) 的混合后验是 $M$ 个 GM（每个 $p_{k-1}^{(i)}$ 本身可能多分量）的加权和，分量数 **指数增长**；且非线性传播/更新无闭式解。第 3–5 章 IMM、GMIMM、AGMIMM 即在不同环节做近似：**混合密度表示**、**非线性线性化**、**自适应分裂** 等。

第 3.1 节的意义：为近似提供 **精确理论基准**，并指明近似牺牲的部分（混合后验形状、非线性处理精度等）。

## 第 3 章 IMM 近似与 GMRC 聚类合并（Sect. 3.2）

### 3.0 动机：混合后验的分量爆炸（docx §3.2）

式 (34) 的混合后验 $p_{k-1}(x_{k-1} \mid \tau_k=j, Z_{0:k-1})$ 是预测步的关键中间量：把上时刻各模式后验按 $\pi_{ij}$ 加权，作为当前模式 $j$ 下预测的起始分布。

设 $p_{k-1}^{(i)}$ 为 GM（式 36），代入 (34) 得式 (37)：**分量数爆炸**——

- 初始每模式可能仅少量分量；
- 每轮预测-更新循环，混合把 $M$ 个 GM 合成更大 GM，分量数为 $\sum_i L_{k-1}^{(i)}$；
- 再经非线性传播与自适应分裂，分量数可 **指数级** 增长 → 必须近似。

**三种压缩策略**（由简到精）：

| 方法 | 策略 | 输出分量数 |
|------|------|-----------|
| IMM | 混合后验矩匹配为 **单高斯** | L=1 |
| GMIMM | 保留 top-r 分量，其余 **合并为一个** 高斯 | L=r+1 |
| **AGMIMM** | **GMRC 聚类** 最优逼近 | L < L_max |

---

### 3.1 混合后验的指数增长

设各模式后验为 GM（式 36）：

$$
p_{k-1}^{(i)}(x_{k-1}) = \sum_{\ell=1}^{L_{k-1}^{(i)}} \omega_{k-1}^{(\ell,i)}\, N(x_{k-1}; m_{k-1}^{(\ell,i)}, P_{k-1}^{(\ell,i)})
$$

代入式 (34) 得 **式 (37)**：混合后验有 $\sum_i L_{k-1}^{(i)}$ 个分量；逐步递推后分量数 **指数增长**。

需近似为 **式 (38)** 的 $L$ 分量 GM（ $L \lt L_{\max}$）。

**【docx 文字说明 — 式 (36)–(38)】** (36) 为模式 $i$ 的后验 GM——GM 是通用逼近器，可表示有限间断 pdf。(37) 为混合后验 **精确** 式，共 $\sum_i L_{k-1}^{(i)}$ 个分量。(38) 为近似目标：用 $L$ 个分量拟合高分量混合，既要捕捉 **多峰与尾部**，又控制分量数。

### 3.2 IMM：强制单高斯

**式 (39)** — $L=1$，矩匹配得 $m_{k-1 \mid k-1}^{(j)}, P_{k-1 \mid k-1}^{(j)}$。

**局限**（图 2b/c）：混沌传播后真实 pdf 多峰、厚尾；单高斯 **低估尾部**、在低密度区引入伪质量 → 21 天传播后偏差更大。

**【docx 文字说明】** IMM 假设混合后每个模式条件分布仍为高斯，等价于丢弃全部 **非高斯特征**（多峰、偏斜、重尾）。弱非线性、短预测周期下尚可；混沌动力学与 **长观测间隙** 下严重失真（图 2b vs 2a）。

### 3.3 GMIMM：保留 top-r + 合并其余

**式 (40)** — $L=r+1$：保留权重最大的 $r$ 个分量，其余 moment-matching 合并为一个高斯。

**局限**（图 2d/e）：低权重分量可能 **空间上相距很远**，合并后协方差过大；弯曲/多峰分布时合并均值可 **落在真 pdf 支撑之外**。

**【docx 文字说明】** GMIMM 做法：(1) 将 (37) 中分量按权重降序；(2) 保留前 $r$ 个；(3) 其余 $L_{\max}-r$ 个 moment-match 合并为一个；(4) 得 $r+1$ 个分量。直觉：非高斯性主要来自少数高权重分量，低权重可打包成“背景高斯”。但低权重分量可能 **空间分散**，强行合并 → 协方差过大、均值偏离支撑区（图 2d）。

### 3.4 AGMIMM：GMRC 聚类合并

**式 (41)** — 用 **Gaussian Mixture Reduction via Clustering (GMRC)** 将式 (37) 约化为 $L$ 分量 GM。

**GMRC 三步**（论文附录 Algorithm 2–4）：

1. **Preprocessing**：Runnalls 贪心合并得 $L$ 个初始聚类中心。
2. **Clustering**：修正 k-means，按 **KL 散度** 将原分量分配到最近中心。
3. **Refinement（可选）**：Newton–Raphson 最小化 NISD；本文为降复杂度 **省略**。

**动机对比**：

| 合并策略 | 依据 | 问题 |
|----------|------|------|
| GMIMM | 权重排序 | 忽略空间结构 |
| **GMRC** | KL 聚类 + 矩匹配 | 保留空间分离的模态，更接近真 pdf（图 2f/g） |

**性质**： $L \to L_{\max}$ 时 AGMIMM $\to$ 精确混合； $L=1$ 时退化为 IMM。

**【docx 文字说明 — GMRC】** 式 (41) 的 $(\omega, m, P)$ **不是** “保留前几个+合并其余”，而是通过 GMRC 得到最优 $L$ 分量近似：

1. **预处理**：Runnalls 基于 KL 的贪心合并得初始 $L$ 分量。
2. **聚类**：将 (37) 的 $\sum_i L^{(i)}$ 个分量按 KL 等距离分配到 $L$ 个簇，每簇一个输出分量。
3. **优化（可选）**：Newton–Raphson 最小化 NISD；本文为速度 **省略**。

GMRC **不是简单截断**：把原分布划为 $L$ 个概率意义上相近的簇，每簇一高斯代表，从而保留多峰与尾部，避免 GMIMM 把空间分离的低权重分量 **强行合并成发散成分**（图 2f）。

### 3.5 GMRC 算法概要（论文附录）

```text
Algorithm 2  GMRC — Gaussian Mixture Reduction via Clustering

Preprocessing:
  用 Runnalls 贪心 KL 合并，将式 (37) 的 sum_i L^{(i)} 个分量
  初步减至 L 个中心 {omega, m, P}_ell

Clustering (Algorithm 3):
  初始化 assignment 矩阵 C (J x L)
  repeat until convergence:
    对每个原分量 j，按 KL 距离 d( N_j, center_ell ) 分配到最近簇
    对每个簇 ell，用 Algorithm 4 计算 barycenter (omega, m, P)

Output:
  L 分量 GMM 近似式 (41)
```

**Algorithm 4（barycenter）直觉**：簇内所有分量按 KL 加权矩匹配，得到新的 $(\omega, m, P)$，使簇内总质量与一阶、二阶矩一致。

**为何优于 GMIMM 的 top-r 合并？**

- GMIMM：丢弃低权重分量信息 → 合并体协方差膨胀、均值可能 **偏离支撑**。
- GMRC：按 **分布形状相似性（KL）** 聚类 → 保留空间分离的多峰结构，更接近式 (37) 的真 pdf（图 2 Monte Carlo 对比）。

## 第 4 章 GMM 预测与状态相关检测更新（Sect. 3.3）

### 4.0 本节目标（docx §3.3）

3.2 节经 GMRC 得到混合后验近似 GM（式 41）。本节完成 **预测** 与 **多传感器更新**，获当前后验 GM。核心挑战：

1. **非线性预测**： $f$ 非线性，高斯经 $f$ 传播后一般非高斯。
2. **状态相关检测概率**： $p_D(x)$ 与位置、FoV、遮挡、照明有关，且在视场边界 **不连续**。
3. **负信息**： $Z=\emptyset$ 同样提供信息，须融入 Bayes 更新。
4. **序贯多传感器更新**：利用条件独立，依次应用 $\Psi_k^{(o,j)}$。

核心思想：**全用 GM 表示密度**；对 $f,h$ 局部线性化（统计线性化/UT）；对 $p_D$ **零阶 Taylor**（分量内近似常数）；通过 3.4/第 5 章 **自适应分裂** 保证线性化精度。

### 4.1 预测步 GM 近似

**式 (42)** — 精确预测（卷积积分，非线性下无闭式）：

$$
p_{k \mid k-1}^{(j)}(x_k) = \sum_{\ell} \omega_{k-1 \mid k-1}^{(\ell,j)} \int N(x_k; f(x_{k-1}), Q_{k-1}^{(j)})\, N(x_{k-1}; m_{k-1 \mid k-1}^{(\ell,j)}, P_{k-1 \mid k-1}^{(\ell,j)})\, dx_{k-1}
$$

**式 (43)–(46)** — mixand 协方差足够小时矩匹配：

$$
p_{k \mid k-1}^{(j)}(x_k) \approx \sum_{\ell} \omega_{k \mid k-1}^{(\ell,j)}\, N(x_k; m_{k \mid k-1}^{(\ell,j)}, P_{k \mid k-1}^{(\ell,j)})
$$

$$
\omega_{k \mid k-1}^{(\ell,j)} = \omega_{k-1 \mid k-1}^{(\ell,j)}
$$

$$
m_{k \mid k-1}^{(\ell,j)} = \int f(x_{k-1})\, N(x_{k-1}; m_{k-1 \mid k-1}^{(\ell,j)}, P_{k-1 \mid k-1}^{(\ell,j)})\, dx_{k-1}
$$

$$
P_{k \mid k-1}^{(\ell,j)} = \int (f(x_{k-1}) - m_{k \mid k-1}^{(\ell,j)})(\cdot)^{\top}\, N(\cdot)\, dx_{k-1} + Q_{k-1}^{(j)}
$$

**关键假设**： $P_{k-1 \mid k-1}^{(\ell,j)}$ 足够小，使 $f$ 在分量支撑内近似线性 → **这是 AGM 分裂的根本动机**（第 5 章）。

**【docx 文字说明 — 式 (42)–(46)】**

- **(42) 精确预测**：每个分量 $(m, P)$ 经 $f$ 加噪声后一般非高斯；若 $P$ 足够小， $f$ 在 $m$ 附近近似线性，输出仍近高斯。
- **(44) 权重不变**：预测不改变相对权重（更新步才改）。
- **(45) 预测均值**： $\text{E}[f(x)]$，用 UT 或统计线性回归（sigma 点加权平均）近似。
- **(46) 预测协方差**：非线性传播不确定性 + 过程噪声 $Q^{(j)}$。

**关键**：每个分量 $P$ 须足够小，使 $f$ 在支撑内近似线性；否则高斯近似误差大——这正是 3.4 节分裂要解决的问题。

### 4.2 RFS 测量似然与负信息

**式 (48a)** — 未检测（ $Z_k^{(o)} = \emptyset$）：

$$
g_k^{(o,j)}(Z_k^{(o)} \mid x_k) = 1 - p_D^{(o)}(x_k; S^{(o)})
$$

**式 (48b)** — 有检测（ $Z_k^{(o)} = \{z_k\}$）：

$$
g_k^{(o,j)}(Z_k^{(o)} \mid x_k) = p_D^{(o)}(x_k; S^{(o)}) \cdot g_k^{(o,j)}(z_k^{(o)} \mid x_k)
$$

其中 $g_k^{(o,j)}(z \mid x) = N(z; h^{(o)}(x), R_k^{(j)})$。

**式 (49)** — 更新后 pdf 结构（未归一化； $I_{\emptyset}$ 为未检测指示）：

$$
[\Psi_k^{(o,j)} p^{(j)}](x) \propto I_{\emptyset} \sum_{\ell} (1-p_D) \omega^{(\ell,j)} N(x; m^{(\ell,j)}, P^{(\ell,j)}) + (1-I_{\emptyset}) \sum_{\ell} p_D \omega^{(\ell,j)} g(z \mid x) N(x; m^{(\ell,j)}, P^{(\ell,j)})
$$

**直觉**：

- **有测量**：似然 × 先验，类似非线性 Kalman 校正。
- **无测量**：权重乘以 $(1-p_D(m))$ → 在 FoV 内/照明满足处 **降低** 该假设的权重 → **负信息**。

**【docx 文字说明 — 式 (48)–(49)】** 量测为空或未检测到：似然为 $(1-p_D(x))$，通过 $p_D$ 传递负信息——某区域 $p_D$ 高却未检测到，该区域后验应降低。式 (49) 为 Bayes 直接应用（正比形式）；因 $p_D$ 与 $h$ 非线性，无法保持 GM，故在 **单个分量支撑内** 设 $p_D \approx p_D(m)$， $h$ 可线性化（EKF/UT）。

### 4.2.1 RFS 似然 → Bayes 更新的推导

**Step 1**：单分量先验 $p^{(j)}(x) = \omega^{(\ell,j)} N(x; m, P)$。

**Step 2**：代入式 (48)。若 $Z=\emptyset$，似然 $= 1-p_D(x)$；后验 $\propto (1-p_D(x))\, p(x)$。

**Step 3**：对 $p_D$ 在 $m$ 处常值近似： $(1-p_D(x)) \approx (1-p_D(m))$，得式 (59) 权重缩放。

**Step 4**：若有测量 $Z=\{z\}$，似然 $= p_D(x)\, N(z; h(x), R)$；在 $m$ 处展开 $p_D$，在 $m$ 附近对 $h$ 统计线性化 → 式 (51)–(53) 与 Kalman 形式同构，但 **权重** 含 $p_D(m)\, q(z)$ 项（检测门控）。

**Step 5**：多分量时对每个 $\ell$ 独立做 Step 3–4，再归一化所有 $\omega_{+}^{(\ell,j)}$。

> **与标准 Kalman 的区别**：(1) 权重随检测概率变化；(2) 空测量也更新权重（负信息）；(3) GMM 需对每个分量重复。

**【docx 文字说明 — 式 (50)–(61)】**

- **有检测** $Z=\{z\}$：标准非线性更新（式 50–53, 55–57）；权重含 $p_D(m)\, q(z)$（检测门控）。
- **无检测** $Z=\emptyset$： $\omega_{+} \propto \omega(1-p_D(m))$，**均值协方差不变**。

负信息的典型效果：不改变连续状态分布 **形状**（无量测值），但调整不同区域 **相对权重**——在 $p_D$ 高却未检测到的分量上惩罚权重。

### 4.3 检测模型线性化与 GM 更新公式

对 $p_D$ 在均值处 **零阶 Taylor 展开** [12]，得 **式 (50)–(58)**（有测量时）：

$$
\omega_{+}^{(\ell,j)} \propto \omega^{(\ell,j)}\, p_D^{(o)}(m^{(\ell,j)}; S^{(o)})\, q_{+}^{(\ell,j)}(z_k^{(o)})
$$

$$
m_{+}^{(\ell,j)} = m^{(\ell,j)} + K_{+}^{(\ell,j)}(z_k^{(o)} - \hat{z}_{+}^{(\ell,j)}), \quad P_{+}^{(\ell,j)} = \text{Joseph 形式 (53)}
$$

$$
q_{+}^{(\ell,j)}(z) = N(z; \hat{z}_{+}^{(\ell,j)}, P_{z,+}^{(\ell,j)}), \quad K_{+}^{(\ell,j)} = C_{+}^{(\ell,j)} (P_{z,+}^{(\ell,j)})^{-1}
$$

$\hat{z}_{+}, C_{+}, P_{z,+}$ 由 **式 (55)–(57)** 的统计线性化积分给出。

**无测量**（式 59–61）： $\omega_{+}^{(\ell,j)} \propto \omega^{(\ell,j)}(1-p_D(m^{(\ell,j)}))$，均值协方差不变。

### 4.4 模式概率更新与后验混合

**式 (62)** — 模式概率的单传感器更新（分空/非空测量两种情况）。

**式 (63)–(64)** — 所有传感器更新后的 mode-conditioned 后验与全概率：

$$
p_k^{(j)}(x_k) \approx \sum_{\ell} \omega_k^{(\ell,j)}\, N(x_k; m_k^{(\ell,j)}, P_k^{(\ell,j)}), \quad p_k(x_k \mid Z_{0:k}) = \sum_{j=1}^{M} \mu_k^{(j)}\, p_k^{(j)}(x_k)
$$

**【docx 文字说明 — 式 (62)–(64)】** 单传感器更新同时改变 GM 与 **模式概率** $\bar{\mu}^{(j)}$：分子含边缘似然预测积分（与式 50 同构），分母为所有模式分子之和。模式概率反映各模式对当前量测/未检测的 **边缘似然**——某模式分量在 $p_D$ 高区未检测，其边缘似然降、模式概率降，滤波器可 **自动识别** 当前运动模式。多传感器序贯更新 $p_k^{(j)}$ 与 $\mu_k^{(j)}$；最终 $p_k(x) = \sum_j \mu_k^{(j)} p_k^{(j)}(x)$。

**本节核心思想小结**：

| 环节 | 做法 |
|------|------|
| 非线性预测 | 各分量独立高斯近似传播（UT/线性化）；要求 P 小 → 第 5 章分裂 |
| p_D | 分量内常数（零阶）→ 负信息 = 权重缩放 |
| 负信息 | 未检测不改变形状，降低 p_D 高分量权重（含模式概率） |
| 序贯更新 | 逐传感器，避免联合高维积分 |
| 模式概率 | 边缘似然竞争，识别机动/弹道 |

这些近似在合理分裂设计下，可处理混沌三体问题中的 **长观测间隙** 与 **未知机动**。

### 4.5 单步更新流程

**【docx 文字说明】** 实现上预测与更新步内均可调用 $\tilde{G}_c$；模式概率 $\mu^{(j)}$ 须与 GMM **同步** 序贯更新。

```text
输入：混合后验 p_{k-1}(x | tau_k=j) 的 GMM，模式先验 mu_bar_{k \mid k-1}^{(j)}
  ↓
预测：对每个模式 j、每个分量 -> m_{k \mid k-1}, P_{k \mid k-1}（+ 分裂，见第5章）
  ↓
for o = 1..O:
    更新 GMM：Psi_k^{(o,j)} -> 式 (50)-(61)
    更新模式概率：Psi_k^{(o,j)} on mu_bar -> 式 (62)
  ↓
输出：p_k^{(j)}(x), mu_k^{(j)}；后验混合 p_k(x)
```

## 第 5 章 AGM 分裂准则（Sect. 3.4 + Sect. 4）

### 5.0 为什么需要分裂？（docx §3.4）

3.3 节多次假设每个高斯分量协方差足够小，从而保证：

1. $f$ 在分量支撑内近似线性（预测）；
2. $p_D$ 在分量支撑内近似常数（更新）；
3. $h$ 可线性化（更新）。

实际 CR3BP 中，**长观测间隙** 使协方差自然增长；**强非线性** 使小协方差迅速“拉长、弯曲”；**视场/遮挡边界** 引入不连续性——线性化假设失效。

**分裂思想**：将大协方差、可能非线性的一个高斯分量，替换为若干 **更小协方差** 子分量的加权和，使各子分量更精确通过非线性变换，提高整体滤波精度。

### 5.0.1 分裂算子（式 65–72）

**式 (65)–(66)** — 分裂算子 $\tilde{G}_c$：

$$
\tilde{G}_c[p(x)] = \prod_{i=1}^{I} G_c[\omega^{(i)} N(x; m^{(i)}, P^{(i)})]
$$

若 c(ω, m, P) ≤ 0，保留原分量；否则拆成 R 个子分量并递归（式 66）：

$$
G_c[\omega^{(i)} N(\cdot)] = \sum_{\ell=1}^{R} G_c[\omega^{(i,\ell)} N(x; m^{(i,\ell)}, P^{(i,\ell)})]
$$

对新生分量 **递归** 应用直至无分量满足分裂准则。

**式 (67)–(72)** — 预生成 **一维标准高斯** 最优分裂库，经缩放、平移、白化推广到多元沿 **u** 方向分裂。

**【docx 文字说明 — 分裂算子】** $c \le 0$ 不分裂； $c \gt 0$ 拆成 $R$ 个子分量并 **递归** 检查——类似树形划分，直至全部分量满足局部线性化门槛。

**【docx 文字说明 — 分裂参数 (67)–(72)】**

- **(67) 标准正态库**：离线优化 $\tilde{\omega}^{(\ell)}, \tilde{m}^{(\ell)}, \tilde{\sigma}$；子分量同 $\tilde{\sigma}$、均值错开。
- **(69)** $\omega^{(i,\ell)} = \tilde{\omega}^{(\ell)} \omega^{(i)}$。
- **(70)** $m^{(i,\ell)} = m^{(i)} + \tilde{m}^{(\ell)} \sigma_u^{(i)} u$， $\sigma_u = (u^{\top} P^{-1} u)^{-1/2}$。
- **(71)** 从 $P^{(i)}$ 减去沿 $u$ 的方差，子分量在 $u$ 方向更窄。
- **(72)** 分裂前后 **一、二阶矩近似守恒**。

直觉：沿方向 $u$ 把高斯切成几片，更窄、错开，捕捉非线性沿 $u$ 的变化细节。

### 5.1 统计线性化误差准则（Sect. 4.1）

**动机**：式 (45)–(46)、(55)–(57) 的线性化/统计线性化只在均值 **小邻域** 内准确；混沌动力学或大协方差分量违反假设。

**式 (73)–(75)** — 统计线性回归： $y = g(x) \approx Gx + b$，最小化 $\text{E}[ee^{\top}] = P_e$。

**式 (76)–(78)** — 分裂准则 [53]：

$$
c_{\text{SL}} = s - s_{\max}, \quad s = \omega^{\Gamma}(1 - e^{-\epsilon})^{1-\Gamma}, \quad \epsilon = \text{tr}(P_e)
$$

- $\Gamma \in [0,1]$：权重 vs 线性化误差的权衡； $s \gt s_{\max}$ 则分裂。
- $g$ 可取 $f$（动力学）或 $h$（测量）。

**【docx 文字说明】** 统计线性化误差准则评估 $g(x) \approx Gx+b$ 的误差； $s \gt s_{\max}$ 则分裂。 $g$ 取 $f$ 或 $h$。

**分裂方向 — 式 (79)–(82)**：方向 $u$ 的非线性度量取 Hessian 的 Frobenius 范数在 $u$ 上的投影，反映 $g$ 沿 $u$ 的 **曲率**。

**【docx 文字说明 — 方向优化】** 约束 $u^{\top} P^{-1} u = 1$ 将 $u$ 限制在 **1σ 椭球**（Mahalanobis 距离为 1），避免选概率极低远区。白化 $u = Sy$， $\|y\|=1$，化为 Rayleigh 商； $y^*$ 为 $S^{\top}(\sum_i D_i^{\top} D_i) S$ 的 **最大特征向量**， $u^* = Sy^*$。在概率显著区域内找 **弯曲最厉害** 的方向分裂，最大程度减少线性化误差。

**何时用 $g=f$ vs $g=h$？**

| 阶段 | g | 准则 |
|------|---|------|
| 预测后 | f（动力学流） | c_SL(f), c_JC |
| 测量更新后 | h（角度模型） | c_SL(h) |

### 5.2 Jacobi 常数方差准则（Sect. 4.2）

**动机**：弹道轨迹上 $C$ 恒定；分量内 $C$ 方差大 $\Rightarrow$ 含不同轨道族 $\Rightarrow$ 传播后协方差膨胀（图 3）。

**式 (83)–(84)**：

$$
\sigma_C^2 = \int (C(r,v) - \text{E}[C])^2\, N(x; m, P)\, dx
$$

$$
c_{\text{JC}} = \min(\sigma_C^2 - \sigma_{C,\max}^2,\; \omega - \omega_{\min})
$$

**实现策略**：Jacobi 准则 **廉价**，用于 **筛选** 待分裂分量；真正分裂方向仍用 5.1 的全非线性度量（动力学 $f$）。

**【docx 文字说明】** Jacobi 方差廉价，用于 **筛选**；分裂方向仍用 5.1 全非线性度量（ $f$）。Sec. 4.2 详述；与图 3 中不同 $C$ 值导致协方差膨胀的现象一致。

### 5.3 负信息 / FoV 边界分裂（Sect. 4.3）

**动机**：式 (50)–(51) 假设 $p_D$ 在分量支撑内 **近似常数**；大分量横跨 FoV 边界时 $p_D$ 剧烈变化 → 近似失效。

**方法**：几何 **递归分裂** [11,12]，直至分量不再显著 overlap FoV 不连续边界（图 4：分裂后 pdf 在 FoV 边界处呈现清晰结构）。

**【docx 文字说明】** 当分量跨越 $p_D$ 剧烈变化区（FoV 边界），零阶 $p_D$ 近似失效；递归分裂直至各子分量上 $p_D$ 近似常数（图 4）。详见 [12]。

### 5.4 非线性约束（Sect. 4.4）

**Jacobi 界** — **式 (85)**（已知 $\Delta V$ 容量时）：

$$
C_{\min}(r_0, v_0+\Delta V) \le C(r,v) \le C_{\max}(r_0, v_0-\Delta V)
$$

违反界的分量 **剪枝**；未知 $\Delta V$ 时用保守 $C_{\min}, C_{\max}$（Table 2）。

**月球碰撞约束**：近月分量递归分裂 + 截断月球内部密度（图 5）。

**【文字说明】** 除 Jacobi 界外，还可注入 **硬约束**：目标不应位于月球实体内部。做法与负信息类似——在近月区分裂分量，把落入月球半径内的概率质量 **截断丢弃**（图 5b）。Jacobi 界（图 5a）则反映 $\Delta V$ 容量：超过一定 $C$ 变化意味着机动过大，可在滤波中剔除不合理分量。二者都是 **软/硬先验** 进入 GMM 的示例，体现 AGMIMM 框架的可扩展性。

### 5.5 目标模式与离散过程噪声（Sect. 4.5）

- **模式 1（弹道）**： $Q^{(1)}$ 小（ $q_w \sim 10^{-3}\,[\text{LU}^2/\text{TU}^3]$）
- **模式 2（机动）**： $Q^{(2)}$ 大（ $q_w \sim 1$）→ 用 **过程噪声** 近似未知推力，而非显式 $\Delta v$ 参数

**【文字说明】** 本文不直接在状态中加入推力参数，而用 **JMS 双模式 + 不同 $Q$** 描述机动：弹道模式过程噪声小，机动模式过程噪声大，等价于“允许状态在一步内发生较大随机加速度”。真实机动是脉冲 $\Delta v$，与连续白噪声 **模型不匹配**；论文故意保留这种 mismatch 以检验滤波器鲁棒性——ASNEES 略保守（ $\approx 0.4$）部分源于滤波器假设“每步都可能机动”。

**式 (86)–(90)** — 连续噪声 $\Gamma Q^{(j)} \Gamma^{\top}$ 映射到离散 $Q_{k-1}^{(j)}$：

$$
Q_{k-1}^{(j)} = \int_{t_{k-1}}^{t_k} \Phi(t_k,t)\, \Gamma Q^{(j)} \Gamma^{\top}\, \Phi^{\top}(t_k,t)\, dt
$$

本文与式 (6) **耦合数值积分** 式 (89) $\dot{Q} = FQ + QF^{\top} + \Gamma Q^{(j)} \Gamma^{\top}$ 求解。

**Table 2 模式转移**： $\pi_{11}=0.99, \pi_{12}=0.01, \pi_{21}=0.1, \pi_{22}=0.9$。

### 5.6 正则化动力学（Sect. 4.6）

**动机**：CR3BP 在地球/月球质心处有奇点；长间隙近月传播需数值稳定。

**式 (91)–(96)** — 虚拟时间 $\gamma$、K–S 变换、8 维增广状态；**式 (97)** 同步传播 $Q^{(j)}$。

Dormand–Prince 5(4) 积分；边界 Jacobi 常数一致性 $\lt 10^{-12}$。

**正则化直觉**：近月时 $r_{\text{Moon}}$ 小 → 式 (91) 使 $dt/d\gamma = r_{\text{Moon}}$ 变小 → 仿真时间在近月区 **放慢** → 积分步长相对物理时间更细 → 避免奇点邻域数值爆炸。

**【文字说明】** 长观测间隙内，滤波器需多次 **纯预测**（无测量）。若数值积分在月球附近发散，GMM 均值与协方差都会失真，后续即使再分裂也无法挽回。K–S 正则化把近月动力学改写到无奇点坐标，并与 $Q^{(j)}$ 的 Lyapunov 方程同步积分（式 97），保证 **传播 + 过程噪声映射** 在同一稳定框架下完成。这是地月 CSO 跟踪实现层面容易被忽视、但对 AGMIMM 成功至关重要的细节。

### 5.7 AGMIMM 单步算法流程（Algorithm 1）

**【docx 文字说明 — 分裂在算法中的位置】** 分裂在 **多个位置** 发生：

1. **预测后**：对 $p_{k \mid k-1}^{(j)}$ 应用 $\tilde{G}_c$（ $c_{\text{SL}}(f)$, $c_{\text{JC}}$ 等）。
2. **每个传感器更新前**：对 $p^{(j)}$ 再分裂（FoV/负信息）。
3. **最终后验**：对所有模式 $p_k^{(j)}$ 再分裂（ $c_{\text{SL}}(h)$ 等）。

每次非线性操作前后都检查分量是否需分裂，保证整个循环中协方差始终小到满足 **局部线性化** 假设。

```text
Algorithm 1  AGMIMM Filter (one time step k, general O sensors)

1. // 交互混合 (IMM)
   For each mode j = 1..M:
     Construct mixed density (34) from {p_{k-1}^{(i)}, mu_{k-1}^{(i)}}
     Approximate via GMRC -> L-component GMM (41): {omega, m, P}_{k-1 \mid k-1}^{(.,j)}

2. // 预测 + 动力学分裂
   For each mode j, each component ell:
     Propagate (43)-(46) with regularized CR3BP
     Apply splitting G~_c with criteria: c_SL (f), c_JC, constraints

3. // 多传感器序贯更新
   mu_bar^{(j)} <- mu_{k \mid k-1}^{(j)} from (27)
   For o = 1..O:
     Apply FoV/negative-info splitting on p^{(j)} if needed
     GMM measurement update (50)-(61) on p^{(j)}
     Mode prob update (62) on mu_bar^{(j)}

4. // 测量后分裂 + 剪枝
   Apply c_SL (h), Jacobi/Moon constraints; prune invalid components

5. // 输出
   p_k^{(j)}(x), mu_k^{(j)};  p_k(x) = sum_j mu_k^{(j)} p_k^{(j)}(x)
   State estimate: E[x_k] under p_k
```

## 第 6 章 总结与速查

### 6.1 三句话总结

1. AGMIMM = **IMM 模式交互** + **各模式 GMM** + **GMRC 混合合并** + **多准则 AGM 分裂**。  
2. 面向地月 **仅角度多传感器 RFS**，系统融合 **负信息、FoV、Jacobi 与碰撞约束**。  
3. 关键在 **局部线性假设**：通过分裂保持小分量，通过 GMRC 保持混合后验形状，通过 JMS 处理未知机动。
